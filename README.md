关于CC0的定义，有一连串专业术语，但你可以简单理解为免费的正版素材。

换句话说，只要标记为CC0素材，无论是视频、音乐还是图片，都不需要付钱，可以直接用，100%不会侵权。

对于视频创作者、新媒体运营者来说，这都是非常值得收藏的。
